# 🧪 Performance Test — Frontend Web
### HTML · CSS · JavaScript

> Project developed as a performance assessment for the frontend web development module at **RIWI**.

---

## 📋 Description

This project involves building a responsive web page that puts into practice the knowledge acquired in semantic HTML, CSS design, and basic DOM manipulation with JavaScript.

---

## 🚀 Technologies used

| Technology | Purpose |
|---|---|
| **HTML5** | Semantic content structure |
| **CSS3** | Styles, layouts (Flexbox / Grid), animations |
| **JavaScript** | Interactivity and DOM manipulation |

## ✅ Features

- [x] Semantic HTML structure (`header`, `main`, `section`, `footer`)

- [x] Basic interactivity with **JavaScript** (DOM events)
